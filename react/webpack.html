webpack

webpack

Webpack은 JavaScript 애플리케이션을 위한 정적 모듈 번들러(Static Module Bundler)입니다.

webpack 개념

Node.js 설치
https://nodejs.org/ko

웹 애플리케이션에서 사용되는 여러 개의 JavaScript 파일, 스타일시트, 이미지 등과 같은 모든 종속성을 하나로 번들링하여 웹 페이지를 구성하는 데 사용됩니다. Webpack은 애플리케이션을 구성하는 모든 모듈과 해당 종속성을 그래프 형태로 분석하고, 각 모듈을 번들로 패키징합니다. 이렇게 번들된 파일은 웹 페이지에 포함되어 사용자의 브라우저에서 실행됩니다. 번들링된 파일은 최적화되고 압축되어 네트워크로부터 효율적으로 전송되며, 필요한 모듈만 로드하여 초기 로딩 시간을 최소화합니다. Webpack은 다양한 모듈 시스템(CommonJS, AMD, ES Modules 등)을 지원하며, JavaScript뿐만 아니라 CSS, 이미지, 폰트 등 다양한 유형의 파일도 모듈로 취급할 수 있습니다. 또한, 코드 스플리팅(Code Splitting), 로더(Loader)를 통한 전처리 및 변환, 플러그인(Plugin) 시스템을 통한 확장성 등 다양한 기능을 제공합니다. Webpack은 현대적인 웹 개발에서 널리 사용되며, React, Vue.js, Angular 등과 함께 사용되는 것이 일반적입니다. 애플리케이션의 구조를 잘 구성하고 효율적인 번들링을 통해 성능을 최적화할 수 있습니다.

Vite

Vite(바이트)는 웹 개발을 위한 최신 빌드 도구(Build Tool)입니다. Vue.js 창시자인 Evan You가 개발한 Vite는 개발 서버와 빠른 번들링을 제공하여 개발자가 더 빠르고 효율적으로 프로젝트를 구축할 수 있도록 돕습니다. Vite는 "단계적으로" 빌드되는 개발 서버를 기반으로 합니다. 이는 개발 중에 빠른 화면 갱신과 빠른 모듈 핫 리로딩(HMR - Hot Module Replacement)을 가능하게 합니다. 일반적인 웹 개발 서버와 달리, Vite는 개별 모듈을 요청 시점에 동적으로 빌드하여 필요한 모듈만 빠르게 전달합니다. 이로써 초기 로딩 속도를 크게 개선하고 개발 과정에서의 생산성을 높일 수 있습니다. 또한, Vite는 ES 모듈(EcmaScript Modules) 기반의 프로젝트를 위한 번들링 도구로 동작합니다. ES 모듈은 모던 브라우저와 Node.js에서 지원되는 표준 JavaScript 모듈 시스템입니다. Vite는 각 모듈을 개별적으로 번들링하여 필요한 모듈만 로드하고 캐시하여 효율적인 개발 환경을 제공합니다. Vite는 기본적으로 Vue.js 프로젝트를 지원하지만, React, Preact, Svelte 등의 프레임워크도 지원하며, 단일 페이지 애플리케이션(SPA) 및 정적 사이트 생성기(Static Site Generator) 프로젝트에도 사용할 수 있습니다. 개발자 친화적인 개발 환경을 제공하면서도 빠른 성능과 효율적인 모듈 관리를 위한 기능을 갖춘 Vite는 현대적인 웹 개발에 많이 사용되는 도구 중 하나입니다.

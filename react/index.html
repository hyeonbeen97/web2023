React

React는 Facebook에서 개발한 JavaScript 라이브러리로, 사용자 인터페이스(UI)를 구축하기 위한 라이브러리입니다. React는 컴포넌트 기반 아키텍처를 사용하여 UI를 구성하며, 데이터의 상태 변화에 따라 자동으로 업데이트됩니다. React는 다른 프레임워크나 라이브러리와 달리 "Virtual DOM"을 사용하여 빠르고 효율적인 UI 업데이트를 지원합니다. 또한, React는 다른 라이브러리와 함께 사용할 수 있는 유연한 구조를 가지고 있어, 개발자들은 필요에 따라 Redux나 React Router 등과 함께 사용할 수 있습니다. React는 SPA(Single Page Application)와 같은 모던 웹 어플리케이션 개발에 매우 유용하며, 현재 많은 기업들이 React를 사용하여 대규모 웹 어플리케이션을 개발하고 있습니다.

React-설치

https://nodejs.org/ko

React는 Node.js를 필요로 합니다. Node.js는 JavaScript를 서버 측에서 실행할 수 있도록 해주는 런타임 환경입니다. Node.js를 설치하려면 https://nodejs.org/ko/ 에서 다운로드 및 설치할 수 있습니다. React를 설치하려면 npm(Node Package Manager)을 사용해야 합니다. npm은 Node.js와 함께 설치됩니다. npm이 설치되었는지 확인하려면 콘솔 또는 터미널 창에서 다음 명령어를 실행해보세요.

npx create-react-app react1

2. JSX

JSX는 JavaScript XML의 약자로, React에서 UI를 작성하기 위한 문법입니다. JSX는 JavaScript 코드 안에서 XML과 유사한 구문으로 UI를 작성할 수 있도록 해줍니다.
React에서 UI를 작성할 때, JSX를 사용하여 UI를 선언적이고 직관적으로 작성할 수 있습니다. JSX는 일반적인 JavaScript 문법과 유사하지만, 다음과 같은 XML 형태의 문법을 사용하여 UI 요소를 생성합니다.

01. if문

조건문은 프로그래밍에서 가장 기본적이면서 중요한 구문 중 하나입니다. 조건문은 주어진 조건이 참(True)인 경우에만 코드 블록을 실행합니다. 만약 조건이 거짓(False)이라면 실행되지 않습니다.


    if(0){
        document.write("실행되었습니다.(true)")
    } else {
        document.write("실행되었습니다.(false)")
    };
결과 확인하기
실행되었습니다.(false)

false 나오는 값, true 나오는 값

false : 0 , null , undefined , false , ""(빈문자열) true : 1, "0", "1" "ABC" "배열[]" "객체{}" ,true

숫자, 문자 , 배열 ,객체 //모두 true가 나오며 0은 제외

02. if문 생략

위 if문을 아래 if문으로 생략하여 쓸수있다.


    const num = 100;

    if(num) {
        document.write("실행되었습니다.(true)")
    } else {
        document.write("실행되었습니다.(false)");
    };

    if(num) document.write("실행되었습니다.(true)");
    else document.write("실행되었습니다.(false)");
결과 확인하기
"실행되었습니다.(true)"

03. 삼항연산자

위의 코드와 아래 코드는 같지만 더 효율적이게 쓰기 위해서는 아래의 삼항연산자 방식으로 쓴다.


    const num = 100;

    if(num == 100){
        document.write("true");
    } else {
        document.write("false")   
    };  

    (num == 100) ? document.write("true") : document.write("false");
결과 확인하기
실행되었습니다.(false)

더 효율적인 방법

뒷내용을 읽지 않아 효율적이다.

04. 다중 if문

if문을 다중으로 사용할 수 있다.


    const num = 120;

    if(num == 90){
        document.write("실행되었습니다(num == 90)");
    } else if(num == 100){
        document.write("실행되었습니다(num == 100)");
    } else if(num == 110){
        document.write("실행되었습니다(num == 110)");
    } else if(num == 120){
        document.write("실행되었습니다(num == 120)");
    } else {
        document.write("실행되었습니다(num == 90)");
    };
결과 확인하기
실행되었습니다(num == 120)

05. 중첩 if문

if문은 중첩 시킬 수 있습니다.


    const num = 100;

    if(num == 100){
        document.write("실행되었습니다.(1)");
        if(num == 100){
            document.write("실행되었습니다.(2)");
            if(num == 100){
                document.write("실행되었습니다.(3)")
            };
        };
    } else {
                document.write("실행되었습니다.(4)");
    };
결과 확인하기
실행되었습니다.(1)실행되었습니다.(2)실행되었습니다.(3)
